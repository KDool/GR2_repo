# GR2_repo

- Python: version 3.9 <br/>
- Python Library: pandas, numpy, scikit-learn, networkx, metis, matplotlib </br>

- Twitch Dataset: DE_edges.csv, DE_target.csv <br/>
- Generated Dataset: run notebook generated_data.ipynb </br>
